# Image to CSS (or text!)

Have you ever wanted a more convenient way of having a less convenient way of displaying an image on a website that's worse in every possible metric than just linking to a JPEG with an "&lt;img&gt;" element like a normal person? If you said, "Yes!" for some reason, this is for you.

Choose ASCII mode to get a seemingly indecipherable pile of text where every 3 characters represents 1 pixel. ASCII mode is incompatible with images that have transparency, but you can try it anyway.

Play with it here: https://tuxedodemon.com/image-to-css
